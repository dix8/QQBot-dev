## [0.4.0] - 2026-03-06

### 新功能

- **前端实时推送**：新增 Admin WebSocket（`/ws/admin`），仪表盘、日志、Bot 状态等数据通过 WebSocket 实时推送，替代 HTTP 轮询；断连时自动回退到轮询
- **统计数据持久化**：消息统计（小时趋势、群/用户排行）持久化到 SQLite，重启后数据不丢失；支持 7 天 / 30 天历史趋势查询；每 5 分钟自动刷盘
- **通知/告警系统**：Bot 连接/断开、心跳超时、插件自动禁用、登录限流等关键事件实时通知；侧边栏和顶栏通知铃铛 + 未读角标 + 下拉面板；warning/error 级别自动 toast 提示
- **定时任务可视化编辑器**：插件配置中的定时消息支持可视化编辑（添加/编辑/删除任务卡片），支持单次定时和 Cron 表达式两种模式，不再需要手写 JSON
- **Cron 表达式定时任务**：定时消息支持 Cron 表达式（5 段标准格式），内置常用预设，自动解析为中文描述
- **消息模板变量**：定时消息支持 `{time}`、`{date}`、`{weekday}` 等动态变量
- **多群定时消息**：单个定时任务可指定多个目标群号
- **仪表盘时间范围切换**：消息趋势图支持 24 小时 / 7 天 / 30 天三种视图
- **消息记录持久化**：Bot 收到的消息自动存入 SQLite，新增消息记录页面支持按类型/群号/QQ号/内容搜索和分页浏览，自动裁剪超过 5 万条的旧消息
- **操作审计日志**：所有 Web 端管理操作（登录、配置修改、插件管理、Bot 管理等）自动记录到审计日志表，系统设置页面可查看完整操作历史
- **配置备份与还原**：系统设置页面支持一键导出/还原完整系统备份（含所有 Bot 配置、插件配置、插件启用状态、超级管理员设置）
- **日志导出**：日志页面新增 CSV / JSON 导出按钮，支持按当前筛选条件导出，最多导出 1 万条
- **多 Bot 聚合视图**：仪表盘在多 Bot 场景下展示每个 Bot 的头像、名称、在线状态和消息发送量总览卡片

### 测试

- 单元测试覆盖 9 个测试文件 / 126 个用例，涵盖 ConfigService、LogService、AuthService、AuditService、MessageStoreService、MessageBufferService、PluginManager、CommandDetector、密码哈希等核心模块

### 安全

- 插件 README 渲染引入 DOMPurify 防止 XSS 注入
- 连接代理接口对危险 OneBot API 操作（`set_restart`、`set_group_leave` 等）记录警告日志
- `.dockerignore` 排除运行时数据目录（`server/data`、`*.db-shm`、`*.db-wal`），避免敏感数据打入镜像

### 修复

- 修复 ConfigView 配置导入按钮在 `<script setup>` 下 `$refs` 无效的问题
- 修复 GroupsView 页面加载时重复调用群列表接口导致双重错误提示
- 修复删除 Bot 时未清理关联的 `bot_config` 配置记录
- 修复 `reverse-ws` 关闭后仍可启动新服务端的问题（`closed` 标志生效）
- 修复 `EventHandler` 连接断开后 `recentApiSends` 未清理导致的内存泄漏
- 修复登录失败记录 Map 积累空数组的内存增长问题
- 修复日志轮询 `total` 计数与实际显示条数不一致
- 修复示例插件 `utils.js` 中 `seg.data` 可能为 `undefined` 的空值访问
- 修复 `README.md` 中 NapCat 反向 WebSocket 连接地址端口错误（`3000` → `6199`）
- 补充 `NoticeEvent` 接口中常用但缺失的字段（`target_id`、`duration`、`file` 等）

### 优化

- 仪表盘显示实时连接状态指示（绿色脉冲点 + "实时" 标签）
- 插件 `configSchema` 新增可选 `editor` 字段，支持自定义编辑器组件
- 日志推送通过 WebSocket 实时到达前端，无需等待轮询间隔
- Bot 状态变更通过 WebSocket 即时反映，减少不必要的全量拉取
- 数据库 `logs` 表新增 `(level, createdAt)` 和 `(source, createdAt)` 复合索引，提升日志查询性能
- 前端各页面（Bots、Logs、Config）统一展示 Store 错误信息
- 提取 `displayBotName` 公共函数消除多处重复的 Bot 名称显示逻辑
- 消除日志与审计日志的内容重叠：用户管理操作由审计日志专门记录，系统日志仅保留运行相关条目
- API 路由错误响应统一使用 `reply.code()` 风格
- 服务关闭流程加入 `try/catch` 保证 `process.exit(0)` 始终执行
- 插件开发文档补充 `super_admin` 权限说明

## [0.3.0] - 2026-03-06

### 新功能

- **群管理页面**：展示已加入群列表（含真实群头像）、按群开关指令响应、批量启用/禁用、搜索筛选，支持移动端响应式
- **QQ 配置管理指令**：系统插件新增 18 个配置指令，可通过聊天直接管理 Bot（自动回复、黑名单、群过滤、在线时段、频率限制等）
- **插件热重载**：前端「重载插件」按钮 + `reloadPlugin`/`reloadAllPlugins` API，ESM 缓存绕过（`.cache` 目录隔离）
- **示例插件模块化**：拆分为 7 个功能模块（admin、features、fun、info、master、notice、utils），降低单文件复杂度
- **整点报时**：定时消息支持 `"type": "hourly"` 格式，24 条消息写在一个配置项中
- **群开关指令**：`/群开关` 指令可在群内直接切换该群的 Bot 响应状态

### 修复

- 修复 `getBotConfig` 中 selfId（QQ号）与数据库 botId 混用导致主人权限校验失败
- 修复指令检测合并策略中 manifest 声明权限被代码扫描覆盖的问题
- 修复预装插件更新后旧版代码仍被缓存使用（启动时重新同步源文件与指令）
- 修复 `/功能列表` 状态与 Web 端插件配置不一致（读取 manifest 默认值兜底）
- 修复 `/设管理` 设置/取消时措辞不准确（"为"/"的"）
- 修复全员禁言/解禁时错误触发个人禁言通知（`user_id === 0` 特殊处理）
- 移除 `/删除主人` 最后一个主人限制（仅超管可操作）
- 移除不支持的「运气王播报」功能（NapCat 未实现 `ElementType = 9`）

### 优化

- 群过滤默认模式改为白名单（新 Bot 默认不响应群消息）
- 日志过滤默认排除 debug 级别，减少干扰
- 示例插件指令支持 @ 和 QQ 号两种方式指定目标
- `/info` 指令展示 Bot 系统昵称与 QQ 昵称

## [0.2.0] - 2026-03-06

### 新功能

- **示例插件全面增强**：新增 17 个扩展指令（群公告、改群名、设管理、头衔、退群、群列表、群成员、好友列表、查群员、查用户、群荣誉、撤回、群历史、已读、群文件等）
- **数据面板**：消息收发趋势统计、活跃群组/用户 Top5
- **系统级过滤**：用户黑名单、群组黑白名单过滤、在线时段限制扩展至通知/请求事件
- **超管列表指令**：`/超管列表`（仅超管可用）
- **自动同意**：好友申请自动同意、入群邀请自动同意移至系统层
- 前端配置页新增用户黑名单、群组过滤、好友/入群自动同意开关

### 修复

- 修复 ConfigView 重复属性导致 Docker 构建失败
- 防撤回改用消息缓存机制（NapCat 撤回后 `get_msg` 返回空内容）
- 合并转发改为匿名身份（`user_id: 10000`），私聊防撤回分两条发送保留富文本

### 优化

- `/菜单` 指令动态展示（按权限过滤、按功能开关过滤）
- 功能列表加序号，支持 `/开启 1` 序号方式切换
- `isMaster()` 同时检查超管和主人列表
- `/帮助` 简化，插件指令展示交由插件自行实现
- 日志自动刷新默认开启，状态 localStorage 持久化
- 关于页面移除数据库路径显示
- README 更新功能特性与指令说明

## [0.1.0] - 2026-03-05

### 新功能

- **QQBot Web Manager 初始版本**
- 反向 WebSocket 服务端，接收 NapCat 连接（OneBot V11 协议）
- 连接状态管理（在线/离线/异常/重连）
- Bot 配置管理（昵称、自动回复、消息范围、关键词回复、在线时段、频率限制）
- 插件系统：zip 导入、启用/禁用/删除、优先级管理、异常隔离
- **插件资源自动清理**：PluginContext 托管定时器 API，卸载后自动释放资源
- 预装示例插件（基础指令 + 群管理 + 通知处理）
- 日志系统（连接日志/运行日志/插件日志，支持筛选搜索）
- 账号密码登录鉴权（JWT）
- 响应式 Web UI（Vue 3 + Shadcn-vue + Tailwind CSS）
- Docker 单容器部署支持

### 修复

- 修复 Docker 预装插件缺失问题
- 修复 docker-compose 数据卷映射配置
- 修复 GitHub Actions workflow 分支名错误
- 修复 README 仓库链接

### 优化

- 系统状态指令新增 CPU 型号/核心/负载、内存占比、平台信息
- `/关于` 指令展示 QQBot 版本号
- 关于页面项目信息从 API 动态获取
- 插件页面卡片布局响应式重构，移动端友好
